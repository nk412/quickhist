quickhist
=========

**NOTE:** requires `numpy`  
`$ pip install --no-cache-dir -U -r quickhist/requirements.txt`  


Plots the distribution of the input data, and provides some basic stats.

Try:

    for x in $( seq 1 100000 ); do echo $(( ($RANDOM + $RANDOM + $RANDOM) )) ; done  |  quickhist
